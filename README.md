# typescript-tool

This repository is focused on exploring TypeScript tools and utilities.

## Features
- Examples of TypeScript utilities.
- Tooling for TypeScript development.

## Installation
To set up the project, clone the repository:
```bash
git clone https://github.com/bul0ix/typescript-tool.git
```

## Usage
1. Navigate to the project directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the tool:
   ```bash
   npm start
   ```
